Elias Plast — Sitio web básico

Archivos principales:
- index.html
- styles.css

Abrir localmente (macOS):

```bash
open "/Users/jonathandelacruz/Documents/elias plast/index.html"
```

Usar Live Server en VS Code:
1. Abre la carpeta `elias plast` en VS Code.
2. Instala la extensión "Live Server" si no la tienes.
3. Haz clic derecho en `index.html` y selecciona "Open with Live Server" o usa el botón "Go Live".

Notas:
- Si la extensión no funciona, prueba renombrar la carpeta para quitar espacios: `elias_plast`.
- Para añadir imágenes, colócalas en una carpeta `assets/` y actualiza los `src` en `index.html`.
