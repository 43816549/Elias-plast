Instrucciones para añadir el logo proporcionado

Copia la imagen que has adjuntado (el logo en formato PNG) a la carpeta `assets/` dentro del proyecto con el nombre `elias-logo.png`.

Pasos desde macOS:

```bash
mkdir -p "/Users/jonathandelacruz/Documents/elias plast/assets"
# Luego copia el archivo descargado al proyecto, por ejemplo:
# cp ~/Downloads/elias-logo.png "/Users/jonathandelacruz/Documents/elias plast/assets/elias-logo.png"
```

Después, recarga la página (`open index.html` o usa Live Server). El logo aparecerá arriba a la izquierda en la cabecera.
